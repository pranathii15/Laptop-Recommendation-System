LAPTOP RECOMMENDATION SYSTEM 
An intelligent laptop recommendation system developed using Fuzzy Logic (Mamdani Type-1) and a 
custom-built dataset to provide personalized laptop suggestions based on user preferences and 
technical requirements. 
Project Overview 
This project helps users choose suitable laptops by analysing inputs such as: 
• Budget  
• RAM  
• Processor  
• Storage  
• Usage Requirements  
The recommendation process is powered by a Mamdani Type-1 Fuzzy Inference System (FIS) created 
in MATLAB, which evaluates user preferences and generates accurate laptop recommendations. 
The system also provides direct shopping links to Amazon and Flipkart, allowing users to instantly 
explore and purchase the recommended laptops from trusted e-commerce platforms. 
The web application is built using: 
• Python (Flask)  
• HTML  
• CSS  
• JSON Dataset

Project Workflow 
1. Created a custom laptop dataset.  
2. Designed a Mamdani Type-1 Fuzzy Logic model in MATLAB.  
3. Generated fuzzy rules and recommendation logic.  
4. Exported the FIS model.  
5. Developed a Flask-based web application.  
6. Built frontend pages using HTML and CSS.  
7. Integrated recommendation logic with the user interface.  
8. Added direct redirection to Amazon and Flipkart product pages.
